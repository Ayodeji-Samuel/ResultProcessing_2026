Format Guide

Each line should contain:

  1. Course Code (e.g., CSC301)
  2. Course Title
  3. Credit Units (1-6)
  4. Semester (1 or 2)
  5. Status (C, R, or E)

Status Codes:

  • C - Compulsory
  • R - Required
  • E - Elective

Example:

CSC301,Data Structures,3,1,C
CSC303,Operating Systems,3,1,C
CSC305,Computer Architecture,2,1,R
CSC307,Database Systems,3,1,C

Separate fields with commas. No spaces around commas unless part of the title.
