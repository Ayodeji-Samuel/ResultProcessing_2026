# Images Directory

This directory stores uploaded images for the Result Processing System.

## Contents

- `logo.png` - University logo (uploaded via Settings > University Logo)

## Supported Formats

- PNG (recommended for logos with transparency)
- JPG/JPEG

## Recommendations

- Use square images (1:1 aspect ratio) for logos
- Recommended minimum size: 200Ã—200 pixels
- Keep file size under 500KB for optimal performance
