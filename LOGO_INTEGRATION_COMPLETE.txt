The EDSU university logo has been integrated into the Result Processing System.

WHAT'S BEEN CONFIGURED:

   - Responsive design for collapsed/expanded states

2. ✓ Upload Interface
   - Settings > University Logo page ready
   - Accepts PNG, JPG, JPEG formats
   - Validates file types
   - Shows preview before upload

3. ✓ PDF Reports
   - Logo automatically included in PDF headers
   - Used in result sheets and spreadsheets
   - Configurable through system settings

4. ✓ Directory Structure
   - Logo storage: app/static/logos/university_logo.png
   - README file included for documentation

TO COMPLETE THE SETUP:
----------------------

Upload your EDSU logo (the circular blue badge) using ONE of these methods:

METHOD 1 - Web Interface (Easiest):
   1. Start the application: python run.py
   2. Login as HOD
   3. Navigate to: Settings > University Logo
   4. Click "Choose File"
   5. Select your EDSU logo image
   6. Click "Upload Logo"

METHOD 2 - Manual File Copy:
   1. Save your logo as: app\static\logos\university_logo.png
   2. Restart the application

METHOD 3 - Python Script:
   1. Save your logo as: edsu_logo.png (in project root)
   2. Run: python save_logo.py

LOGO SPECIFICATIONS:
-------------------
- Format: PNG (recommended), JPG, or JPEG  
- Size: 200x200 pixels minimum
- Design: Your EDSU circular badge logo
- Background: Transparent PNG works best

WHERE THE LOGO APPEARS:
-----------------------
✓ Sidebar header (left navigation)
✓ PDF result sheets
✓ Spreadsheet exports  
✓ Report headers
