# How to Add the EDSU University Logo

You have **3 options** to add the logo to the system:

## Option 1: Upload via Web Interface (Recommended) ✓

1. Start the application
2. Login as HOD
3. Go to **Settings > University Logo**
4. Click "Choose File" and select your EDSU logo image
5. Click "Upload Logo"

The logo will automatically appear in:
- Sidebar navigation
- PDF reports
- Result spreadsheets

## Option 2: Manual File Placement

1. Save your EDSU logo image as `university_logo.png`
2. Place it in: `app/static/logos/`
3. Refresh the application

## Option 3: Use the Python Script

1. Save your logo as `edsu_logo.png` in the project root
2. Run: `python save_logo.py`

## Logo Specifications

- **Format**: PNG (recommended for transparency), JPG, or JPEG
- **Size**: 200x200 pixels minimum
- **Recommended**: Square or circular design
- **Background**: Transparent PNG works best
